void EXTI0_IRQHandler ( void  )

The EXTI_Line0 interrupt handler.

Note
The interrupt handler checks whether tap is greater than 0. If it is, the value remains the same. If it is not, then tap is set to 13. Combined with the logic in main.c and a 25Hz clock, this creates a timeout of about half a second during which additional taps are not detected.
